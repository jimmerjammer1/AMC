# AMC Compliance Tracker

Track AMC compliance, E&O insurance, and license renewals for appraisers.

## 🚀 Quick Start - Get Online in 5 Minutes

### Step 1: Create GitHub Account
1. Go to [github.com](https://github.com)
2. Click "Sign up" (if you don't have an account)
3. Follow the steps to create your account

### Step 2: Create New Repository
1. Click the **+** icon (top right) → "New repository"
2. Name it: `amc-tracker` (or any name you want)
3. Make it **Public**
4. ✅ Check "Add a README file"
5. Click **"Create repository"**

### Step 3: Upload Your File
1. In your new repository, click **"Add file"** → **"Upload files"**
2. Drag and drop `index.html` into the box
3. Click **"Commit changes"**

### Step 4: Enable GitHub Pages
1. Click **"Settings"** tab (top of your repository)
2. Click **"Pages"** in the left sidebar
3. Under "Source":
   - Select: **main** branch
   - Select: **/ (root)** folder
4. Click **"Save"**
5. Wait 1-2 minutes

### Step 5: Access Your App
Your app will be live at:
```
https://[YOUR-USERNAME].github.com/amc-tracker
```

Example: `https://jimmerjammer1.github.com/amc-tracker`

**That's it! Your app is live and ready to use!** 🎉

## 📱 How to Use

### First Time Setup
1. Go to **"Manage Appraisers"** tab
2. Add your appraisers (James, Matt, John are pre-loaded)
3. Go to **"Manage AMCs"** tab
4. Click **"Add AMC Client"** to add your first AMC

### Track Compliance
1. Go to **"AMC Compliance Dashboard"**
2. Click on any AMC to expand it
3. Check off E&O and License boxes as you complete them
4. AMCs turn green when complete!

### Add Notes to AMCs
1. In **"Manage AMCs"** tab
2. Click **"📝 Notes"** on any AMC
3. Select status (Sent Email, No Response, etc.)
4. Type your note
5. Click **"Add Note"**

## 💾 How Data is Stored

**Your data is saved in your browser's localStorage:**
- ✅ Data persists between sessions
- ✅ Works offline after first load
- ✅ Private to your computer/browser
- ⚠️ Each person's data is separate
- ⚠️ Clearing browser data deletes everything

**To share data across team:**
- Each person uses their own browser storage
- OR export/import data manually (future feature)
- OR set up Firebase (advanced)

## 🔧 Troubleshooting

**App won't load?**
- Make sure index.html is in the root folder
- Check GitHub Pages is enabled
- Wait 2-3 minutes after enabling Pages

**Data disappeared?**
- Did you clear browser cache/cookies?
- Are you using a different browser?
- Try the same browser you used before

**Can't access the site?**
- Check the URL matches your GitHub username
- Repository must be public
- Pages must be enabled in Settings
